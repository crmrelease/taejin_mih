# 간단 계산기 (Calculator) 기획서

> 작성: 솜이 (기획자)
> 작성일: 2026-05-09
> 수신: 랑이 (개발자/메이커)
> 상태: 개발 착수 요청

## 1. 한 줄 요약

웹 브라우저에서 바로 쓰는 **사칙연산용 간단 계산기**를 하나 만들어주세요. 별도 설치 없이 페이지 열면 바로 사용 가능해야 해요.

## 2. 왜 만드나요 (배경)

- 팀이 만든 첫 번째 작은 제품으로 적당해요. 화면도 작고, 로직도 단순해서 **3-에이전트 협업 흐름(솜→랑→모)을 끝까지 한번 돌려보기 좋아요.**
- 너무 복잡한 기능은 빼고, "버튼 누르면 답이 나온다"는 핵심 경험만 깔끔하게 만드는 게 목표예요.

## 3. 사용자 시나리오

평범한 사용자가 브라우저로 페이지를 열어서:

1. 화면에 계산기가 보여요
2. 숫자 버튼을 눌러서 첫 번째 숫자를 입력
3. `+` `-` `×` `÷` 중 하나를 눌러요
4. 두 번째 숫자를 입력
5. `=` 누르면 결과가 화면에 나와요
6. `C` 누르면 처음으로 돌아가요

## 4. 꼭 들어가야 하는 기능 (Must)

| 항목 | 설명 |
|---|---|
| 숫자 입력 | 0~9 버튼, 누르면 화면에 누적 |
| 소수점 | `.` 버튼, 한 숫자에 한 번만 입력 가능 |
| 사칙연산 | `+`, `-`, `×`, `÷` 네 가지 |
| 결과 보기 | `=` 버튼 누르면 계산해서 화면에 표시 |
| 전체 지우기 | `C` 또는 `AC` 버튼, 입력값/연산 초기화 |
| 0으로 나누기 처리 | 결과 자리에 `Error` 같은 문구 표시 (앱이 멈추면 안 돼요) |

## 5. 있으면 좋은 기능 (Nice to have)

- 한 글자만 지우는 백스페이스(`⌫`) 버튼
- 부호 바꾸기(`±`) 버튼
- 키보드로도 입력 (숫자키, `+`, `-`, `*`, `/`, `Enter`, `Backspace`, `Escape`)
- 연속 계산 — `=` 안 누르고 `2 + 3 + 4` 처럼 이어서 누르면 중간 결과를 화면에 보여주기

> Nice to have는 시간 남으면 추가해주세요. 시간 없으면 생략 OK.

## 6. 안 만들어도 되는 것 (Out of scope)

- 공학용 계산(괄호, 제곱근, 삼각함수 등) — 이번엔 빼주세요
- 계산 기록(history) 저장
- 다크 모드, 테마 전환
- 로그인/사용자 데이터 저장
- 모바일 전용 디자인 — 데스크톱 기준이면 충분해요. 깨지지만 않으면 됩니다

## 7. 화면 윤곽 (러프)

```
┌────────────────────┐
│       0            │   ← 디스플레이 (오른쪽 정렬)
├────┬────┬────┬────┤
│ AC │ ⌫  │ ±  │ ÷  │
├────┼────┼────┼────┤
│ 7  │ 8  │ 9  │ ×  │
├────┼────┼────┼────┤
│ 4  │ 5  │ 6  │ -  │
├────┼────┼────┼────┤
│ 1  │ 2  │ 3  │ +  │
├────┼────┼────┼────┤
│ 0       │ .  │ =  │
└────────────────────┘
```

- 디스플레이: 입력 중인 숫자/결과 표시
- 연산자 버튼은 색을 살짝 다르게 (예: 강조색)
- 좋아하는 색이 **보라색**이라고 하셨으니, 강조 버튼은 보라색 계열로 부탁해요 🟣

## 8. 기술 권고 (강제 아님)

- **현재 `index.html` 한 페이지 구조라서, 그 흐름에 맞춰 정적 HTML + JS로 만드는 게 깔끔할 것 같아요.**
- 별도 빌드 도구나 프레임워크는 굳이 안 들여도 되는 규모예요.
- 파일 위치는 랑이가 편한 대로요. 예: `index.html`에 직접 추가하거나, `calculator.html` 새로 만들어서 `index.html`에서 링크 거는 식.
- 더 좋은 방법이 있으면 랑이 판단대로 진행하셔도 됩니다.

## 9. 받아들이는 기준 (Acceptance Criteria)

이 셋이 다 되면 "완성"이에요.

1. **기본 사칙연산 4개가 정확히 계산돼요.** 예: `7 × 8 = 56`, `10 ÷ 4 = 2.5`
2. **0으로 나눠도 페이지가 멈추지 않아요.** 화면에 에러 표시만 뜨고, `C` 누르면 다시 정상 동작
3. **C/AC 누르면 디스플레이가 0으로 초기화돼요.**
